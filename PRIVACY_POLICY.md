# Privacy Policy for Focus Chrome Extension

Last updated: January 2026

Focus is a Chrome extension that helps users manage their tasks by synchronizing with Google Tasks and displaying them in a calendar-based interface.

We respect user privacy and are committed to protecting it. This Privacy Policy explains what information is accessed, how it is used, and how it is handled.

---

## 1. Information We Access

Focus accesses the following information only after explicit user consent via Google OAuth:

- Google Tasks data
    - task lists
    - tasks and their metadata (titles, due dates, statuses)

- Google account email address
    - provided via Google OAuth
    - used solely to identify the authenticated user

Focus does not collect or access:

- personal profiles
- contacts
- messages
- browsing history
- financial or payment information
- location data

---

## 2. How the Information Is Used

The accessed data is used exclusively to provide the core functionality of the extension:

- synchronizing tasks with Google Tasks API
- displaying tasks in calendar and list views
- creating, editing, deleting, and reordering tasks
- moving tasks between dates using drag & drop

The data is never used for:

- analytics
- tracking
- advertising
- marketing
- profiling
- resale or sharing with third parties

---

## 3. Data Storage

- All data related to the extension is stored:
    - locally in the user’s browser using chrome.storage
    - and/or in the user’s Google Tasks account
- Focus does not store any data on external or developer-controlled servers.
- Focus does not perform backups, exports, or cross-device synchronization.

---

## 4. Data Sharing

Focus does not sell, share, or transfer user data to any third parties.

The only data exchange occurs directly between the extension and:

- Google OAuth services
- Google Tasks API

All communication with Google services is performed over secure HTTPS connections using official Google APIs.

---

## 5. Permissions Explanation

Focus requests the following Chrome permissions:

- identity
    - used for Google OAuth authentication
- storage
    - used to store user preferences and cached task data locally

Host permissions:

- https://www.googleapis.com/*
    - required to communicate with Google Tasks API

No additional permissions are requested or used.

---

## 6. Analytics and Logging

- Focus does not use any analytics services.
- Focus does not collect usage statistics.
- Errors and logs are not sent outside of the extension.
- Any logging (e.g. console.log) remains local to the user’s browser.

---

## 7. Data Security

- All communication with Google services is encrypted via HTTPS.
- No external servers are used.
- User data remains under the control of:
    - the user’s browser
    - the user’s Google account

---

## 8. Data Retention and Deletion

- Focus does not retain data outside the user’s browser.
- Users can remove all locally stored data by:
    - signing out of the extension
    - or uninstalling the extension
- Google Tasks data can be viewed or deleted directly via the user’s Google account.

---

## 9. Children’s Privacy

Focus is not intended for children under the age of 13.

The extension does not knowingly collect personal information from children.

Any changes will be reflected on this page with an updated “Last updated” date.
