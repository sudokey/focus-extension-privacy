# Focus - Visual Planner for Google Tasks

A Chrome extension that provides a visual calendar view for managing your Google Tasks with drag & drop functionality.

## Features
- Week and month calendar views
- Drag & drop task planning
- Full Google Tasks synchronization
- Complete privacy - no external servers

## Privacy
All data is stored locally in your browser and synced with your Google Tasks account. No external servers are used.
